# **Lógica de Programação**

Repositório destinado às atividades desenvolvidas na disciplina de : Lógica de Programação - 1Sem / 2026 - Tecnologia de Ciência de Dados

## Estrutura

- Exercícios de fixação e aprendizado - L1
    - Exercício 1 - Operadores Aritméticos
    - Exercício 2 - Operadores Atribuição
    - Exercício 3 - Operadores Comparação
    - Exercício 4 - Operadores Lógicos
    - Exercício 5 - Operadores de Identidade (is, is not)
    - Exercício 6 - Operadores de Associação (in, not in) 
    - Exercício 7 - Estrutura Condicional (IF, ELIF, ELSE)
    - Exercício 8 - Estrutura de Repetição FOR
    - Exercício 9 - Estrutura de Repetição WHILE

- Exercícios de fixação e aprendizado - L2
    - Exercício 1 - Integração: Operadores + Condicionais
    - Exercício 2 - Integração: Repetição FOR + Operadores de Associação
    - Exercício 3 - Integração: Repetição WHILE + Condicionais + Atribuição
    - Exercício 4 - Integração: Identidade + Coleções Simples
 
- Exercícios de fixação e aprendizado - L3
    - Case: Você recebeu uma lista de transações de uma empresa. Seu trabalho é
      criar um script que processe essas transações, verifique a integridade dos dados e
      calcule o saldo final, mas com regras rígidas de segurança.
