### ◼︎ 03 - Exercícios de Fixação e Aprendizado – L3

- ♦︎ **Case:** Aplicação integrada dos temas da L1.  
  O script recebe uma lista de transações de uma empresa, processa os dados, verifica a integridade das informações e calcula o saldo final, seguindo regras rígidas de segurança.

---

### ◼︎ 04 - Desafio: Estudo de Caso – A Crise da Startup TechInvest

- ♦︎ **Case fictício:** Desenvolvimento do “Prime”, um robô-analista em Python para atender clientes e orientar investimentos sem uso de banco de dados.  

  O sistema:
  - Coleta dados financeiros  
  - Analisa a situação do usuário  
  - Identifica o perfil de risco  
  - Sugere investimentos com base em regras condicionais  
  - Projeta objetivos futuros considerando disciplina e tempo de investimento  

---

### ◼︎ 05 - Desafio: Processamento de Dados de Vendas

- ♦︎ Processamento de dados de vendas inseridos pelo usuário, aplicando conceitos de **Listas** e **Tuplas** em Python.

---

### ◼︎ 06 - Desafio: Pipeline de Auditoria para Ciência de Dados

- ♦︎ Desenvolvimento de um pipeline de auditoria que coleta, limpa e analisa vendas de filiais com dados inconsistentes.

  O sistema:
  - Filtra valores inválidos  
  - Calcula faturamento total e média  
  - Classifica a performance da filial  
  - Gera relatório formatado com métricas consolidadas  

  **Requisitos técnicos utilizados:**
  - Funções  
  - while  
  - for  
  - Listas e Tuplas  
  - Condicionais (if/elif/else)  
  - Operadores  

---

### ◼︎ 07 - Exploração do Planeta – Aplicação de Função

- ♦︎ **Case fictício:** Exploração espacial no planeta Alphara-7.

  O programa:
  - Simula decisões da tripulação  
  - Controla obstáculos e descobertas científicas  
  - Utiliza funções, condicionais e repetições  
  - Soma conquistas obtidas  
  - Finaliza a missão com o retorno à nave
